Create api/analyze-v3.js as a new file. This is the Phase 7 analysis endpoint for v3.0 sessions (those with meridian.stateUrl). Structure:

**Imports:**

import { isAllowedBlobUrl } from './analyze.js';
import {
  countIfDepth,

  hasUnguardedDivision,

**parseStateBlob(text) function** (local, not exported):
- Split on '\n', iterate lines
- `parts = line.trimEnd().split('\t')`
- If parts[0] === 'MODULE': push `{ id: parts[1]||'', name: parts[2]||'', prefix: parts[3]||'', lineItems: [] }` to modules array
- If parts[0] === 'CALC'|'INPUT'|'ITEM' and current module exists: push a line item with all fields the normalised shape requires:
  - name: parts[1]||''
  - formatType: parts[2]||''
  - summaryMethod: parts[3]||''
  - formula: parts[4]||''
  - hasFormula: parts[0] === 'CALC' && (parts[4]||'').length > 0
  - isInput: parts[0] === 'INPUT'
  - formulaTruncated: (parts[4]||'').endsWith('…')
  - id: '' (not in blob)
  - dimensions: [] (not in blob)
  - dimensionCount: 0
  - notes: ''
  - formulaLength: (parts[4]||'').length
  - ifDepth: countIfDepth(parts[4]||'')
  - hasSumLookup: hasSumLookup(parts[4]||'')
  - hasHardcodedSelect: hasHardcodedSelect(parts[4]||'')
  - hasUnguardedDivision: hasUnguardedDivision(parts[4]||'')
- Blank line: set current = null

**toNormalized(modules) function** (local):
Returns the object shape buildDependencyGraph() and all analysis-core.js functions accept:
```javascript
{
  modelId: '',
  workspaceId: '',
  partialLoad: false,
  rawModuleCount: modules.length,
  excludedModules: [],
  modules: modules.map(m => ({
    ...m,
    lineItemCount: m.lineItems.length,
    dimensions: [],
    dimensionCount: 0,
  })),
}
```

**Handler export default async function handler(req, res):**

1. applyCors — import from './_cors.js' (same pattern as all other endpoints)
2. OPTIONS → 200, non-POST → 405
3. Read stateUrl from req.body — if missing return 400
4. isAllowedBlobUrl(stateUrl) check — if false return 400 with message 'Invalid stateUrl — must be a Vercel Blob URL'
5. Read evidencePack from req.body (optional, may be absent for older callers) — store as `const evidencePack = req.body?.evidencePack || null`
6. Set SSE headers — BEFORE any await (this is the critical ordering):
   ```javascript
   res.setHeader('Content-Type', 'text/event-stream');
   res.setHeader('Cache-Control', 'no-cache, no-transform');
   res.setHeader('Connection', 'keep-alive');
   res.setHeader('X-Accel-Buffering', 'no');
   res.flushHeaders();
   ```
7. sendEvent helper — same pattern as analyze.js line 208:
   ```javascript
   function sendEvent(obj) {
     res.write(`data: ${JSON.stringify(obj)}\n\n`);
     if (typeof res.flush === 'function') res.flush();
   }
   ```
8. try block:
   - sendEvent({ type: 'stage', stage: 'parsing', label: 'Fetching model state…' })
   - fetch(stateUrl) — throw on non-ok
   - const stateText = await response.text()
   - sendEvent({ type: 'stage', stage: 'classifying', label: 'Parsing model state…' })
   - const modules = parseStateBlob(stateText)
   - if (!modules.length) throw new Error('State blob contains no modules — re-fetch model state')
   - const normalized = toNormalized(modules)
   - sendEvent({ type: 'stage', stage: 'graph', label: 'Building dependency graph…' })
   - // Plans 02 and 03 will fill in the analysis calls here
   - sendEvent({ type: 'complete', version: 'v3', moduleCount: modules.length, lineItemCount: normalized.modules.reduce((s,m)=>s+m.lineItemCount,0) })
9. catch block: sendEvent({ type: 'error', message: err.message })
10. finally: res.end()

Do NOT add exports for parseStateBlob or toNormalized — they are internal to this file.
Do NOT import buildDependencyGraph or other analysis functions yet — Plan 02 adds those.
